# project1-travelling-salesman-problem
O Problema do Caixeiro Viajante (PCV) é um dos problemas clássicos na área de Otimização e pode ser descrito da seguinte maneira. Suponha que você seja um representante comercial e precisa visitar n cidades diferentes para vender seu produto. Nesta viagem, cada cidade deve ser visitada uma única vez e, ao final do trajeto, você deve retornar à cidade de origem. Existem diferentes possibilidades de realizar este trajeto, mas o interesse está no trajeto de menor custo
